## Description
Copilot을 위한 Langchain 기반의 RestAPI 서비스
 - Nest.js
 - Langchain
 - OpenAI
 - LLAMA2(not yet)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start:dev

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
